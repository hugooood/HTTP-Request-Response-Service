httpbin(1): HTTP Request &Response Service

Freely hosted in HTTP, HTTPS, & EU flavors by Kenneth Reitz & Heroku.

ENDPOINTS

  • /This page.
  • /ipReturns Origin IP.
  • /user-agentReturns user-agent.
  • /headersReturns header dict.
  • /getReturns GET data.
  • /post Returns POST data.
  • /patch Returns PATCH data.
  • /put Returns PUT data.
  • /delete Returns DELETE data
  • /encoding/utf8Returns page containing UTF-8 data.
  • /gzipReturns gzip-encoded data.
  • /deflate Returns deflate-encoded data.
  • /brotli Returns brotli-encoded data.
  • /status/:code Returns given HTTP Status code.
  • /response-headers?key=val Returns given response headers.
